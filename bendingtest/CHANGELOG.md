# Changelog

## 0.3.0 - 2026-03-02
- Added dedicated bending specimen geometry vocabulary `sdata-bending-specimens.ttl` (`https://w3id.org/sdata/bending/specimens/`).

## 0.2.0 - 2026-03-02
- Added import of `https://w3id.org/sdata/quantities/`.
- Mapped bending properties to `sdata-quantities.ttl` via `owl:equivalentProperty`.

## 0.1.0 - 2026-03-02
- Initial bending ontology scaffold created.
- Added `BendingTest`, `BendingTestResult`, and core bending KPIs.
